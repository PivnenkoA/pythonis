Создание git repository

Существует два основных подхода для создания репозирория:

  1. Клонирование уже существующего с сервера;
  2. Инициализация уже существующего каталога.

Для инициализации каталога нужно перейти в нужную директорию и использовать простую команду:

$git init

Файлы в репозитории могут находиться в дух состояниях:

  1. Под версионным контролем;
  2. Не отслеживаемые файлы (untracked file).

Для того, чтобы перевести untracket file в состояние контроля версии нужно добавить файл в репозиторий:

$git add fileName

Для добавления всех файлов:

$git add .

Для проверки статуса репозитория используем команду:

$git status

ВСЕГДА ПЕРЕД КОММИТОМ ИСПОЛЬЗУЙ КОМАНДУ git add ЧТОБЫ СОХРАНИТЬ ИЗМЕНЕНИЯ.

Для коммита используем команду:

$git commit -m "Коментарий"
