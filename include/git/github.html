Саздать окаунт в Git Hub не состовляет труда. Создать новый репозиторий тоже не сложно:

  • Нажимаем кнопку "New repository"
  • Водим имя репозитория
  • Вводим описание репозитория
  • Жмем "Creat repository"

Для определения удаленного репозитория вводим:
$git remote add origin https://github.com/UserName/RepositoryName.git
git push -u origin master
