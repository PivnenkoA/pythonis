Git

Является современной системой контроля версий. Git состоит из трех основных частей:

  • Git directory (repository)
  • Working directory
  • Staging area

Работа в git представляет собой следующее:

  • Подготовка файла в рабочей директории (Working Directory);
  • Подготовка макета в облати подготовленных файлов (Staging Area);
  • Помечение файла на постоянное хранение в репозиторий git (Git Directory).

Первое что нужно сделать в git - это настроить имя пользователя и email:

$git config --global user.name "userName"
$git config --global user.email "user@email.com"

Что бы вывести информацию о настройках нужно ввести следующюю команду:

$git config --list

Git имеет очень хорошую документацию. Для ее вызова нужно ввести команду:

$git help

Для более конкретной информации по командам Git:

$git help comand

Или:

$git comand --help
