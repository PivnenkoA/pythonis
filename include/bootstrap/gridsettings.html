Резиновый контейнер

Следующий класс делает наш контейнер адаптивным:

<div class = "container-fluid">Содержимое контэйнера</div>

Отступы в сетке

<div class = "col-lg-5 col-lg-offset-1">Содержимое контэйнера$lt/div>

Спрятать колонку

Если нужно убрать колонку на определенном разрешении то пишем например так:

<div class="col-lg-2 hidden-sm">Содержимое контэйнера</div>
